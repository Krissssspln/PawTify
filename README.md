Pawtify
Descripción del Proyecto

Pawtify es una plataforma web enfocada en la venta de productos para mascotas, permitiendo a los usuarios explorar productos, gestionar perfiles de mascotas, interactuar mediante publicaciones tipo blog y recibir asistencia personalizada mediante un chatbot especializado.

El sistema busca mejorar la experiencia de compra ofreciendo recomendaciones basadas en el perfil de cada mascota y una interfaz moderna, intuitiva y funcional.

Funcionalidades Principales
Registro e inicio de sesión de usuarios
Perfil de usuario personalizado
Registro de mascotas por usuario
Catálogo de productos destacados
Sistema de blog con publicaciones e imágenes
Comentarios y likes en publicaciones
Panel de administración
Recomendaciones de productos
Chatbot veterinario/comercial con integración de Groq + Supabase Edge Functions
